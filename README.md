# The Birth of Distributed Intelligence

### [View the presentation](https://jon-westbrook.github.io/code-and-bourbon-presentation/)

A single-file cyberpunk presentation engine. Zero dependencies. No frameworks, no build tools, no npm. Just HTML, CSS, and JavaScript.

## Features

- **Single-file architecture** — everything lives in `index.html`
- **Keyboard navigation** — arrow keys to move between slides
- **Fullscreen mode** — press `F` to toggle
- **Three-act color theming** — cyan, amber, magenta sections
- **Looping chronograph timer** on the final slide
- **GitHub Pages deployment** — push to main, auto-deploys

## Usage

Open `index.html` in any modern browser. Arrow keys to navigate. `F` for fullscreen.
